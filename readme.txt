** Installation **

1. Download our .Zip file. Go to the WP Dashboard click (Plugins) on left menu -> click (Add New Plugin) button -> click (Upload Plugin) button -> Select our .Zip File and click (Install Now) button. 
2. Activate the plugin through the 'Plugins' menu in WordPress
